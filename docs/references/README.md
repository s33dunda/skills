# References

Use this directory for external references agents need locally: upstream docs snapshots, protocol notes, or vendor constraints relevant to the repo as a whole.

Prefer small, task-relevant excerpts or generated `*-llms.txt` snapshots over large copied manuals. Record the source URL and retrieval date at the top of each reference.

Note: skill-specific references live in `skills/<name>/references/` and travel with the skill's `SKILL.md`. This directory is for repo-level references that span skills or belong to the repo itself.
